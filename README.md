**TrekTales** is a full-stack web application built for travelers to share their journeys, plan trips, and connect with guides.

---

## Features
- Public travel posts with images
- Private travel journal
- Real-time messaging and notifications
- AI-based trip planner
- Guide connect feature to link travelers with local guides

---

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- WebSockets (real-time messaging & notifications)
- Cloudinary (image storage)
- Gemini 2.5 Flash (AI trip planning)

---

## Setup
1. Clone the repository
2. Install dependencies for both client and server
3. Add environment variables (MongoDB, Cloudinary, Gemini API)
4. Run frontend and backend servers

---

## Purpose
TrekTales helps travelers document experiences, plan trips intelligently, and build meaningful connections while traveling.
