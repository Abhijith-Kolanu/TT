**TrekTales** is a full-stack web application built for travelers to share and document their journeys, plan trips, and connect with guides.

---

## Features
- Public travel post sharing
- Private travel journal
- Real-time messaging and notifications
- AI-powered trip planner
- Guide connect feature

---

## Tech Stack
- MERN Stack (MongoDB, Express.js, React.js, Node.js)
- WebSockets
- Cloudinary (image uploads)
- Gemini 2.5 Flash (AI model)

---

## Project Objective
The goal of TrekTales is to provide a single platform where travelers can share and document journeys, communicate in real time, and plan trips using AI.
